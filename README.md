## How to use it

- Build from sources:

```
$ git clone https://github.com/app-generator/eleventy-html5up-paradigm.git
$ cd eleventy-html5up-paradigm
$ yarn # install modules
$ yarn dev # start in development mode

$ # app is running on http://localhost:4000
```
